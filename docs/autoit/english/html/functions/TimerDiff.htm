Function Reference

TimerDiff

Returns the difference in time from a previous call to TimerInit().

TimerDiff ( handle )

Parameters

handle Handle returned from a previous call to TimerInit().

Return Value

Returns the time difference (in milliseconds) from a previous call to TimerInit().

Remarks

Some processors are known to cause the function to return incorrect result.

Related

TimerInit

Example

#include <Constants.au3>

Local $hTimer = TimerInit() ; Begin the timer and store the handle in a variable.
Sleep(3000) ; Sleep for 3 seconds.
Local $iDiff = TimerDiff($hTimer) ; Find the difference in time from the previous call of TimerInit. The variable we stored the TimerInit handlem is passed as the "handle" to TimerDiff.
MsgBox($MB_SYSTEMMODAL, "Time Difference", $iDiff)
