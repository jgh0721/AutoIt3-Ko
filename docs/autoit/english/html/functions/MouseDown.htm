Function Reference

MouseDown

Perform a mouse down event at the current mouse position.

MouseDown ( "button" )

Parameters

button The button to click: "left", "right", "middle", "main", "menu", "primary", "secondary".

Return Value

Success: Returns 1.
Failure: Returns 0, the button is not in the list.

Remarks

See MouseClick remarks.
User responsibility: For every MouseDown there should eventually be a corresponding MouseUp event.

Related

MouseUp, MouseClick, MouseClickDrag, MouseGetPos, MouseMove, MouseCoordMode (Option), MouseClickDownDelay (Option)

Example

MouseDown("left") ; Set the left mouse button state as down.
Sleep(100)
MouseUp("left") ; Set the left mouse button state as up.
