Function Reference

MouseWheel

Moves the mouse wheel up or down.

MouseWheel ( "direction" [, clicks = 1] )

Parameters

direction "up" or "down"
clicks [optional] The number of times to move the wheel. Default is 1.

Return Value

Success: Returns 1.
Failure: Returns 0, the direction is not recognized.

Remarks

None.

Related

MouseClick, MouseClickDrag, MouseGetPos, MouseMove, MouseCoordMode (Option)

Example

MouseWheel("up", 10) ; Move the mouse wheel up ten times.
