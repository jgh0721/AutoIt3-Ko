Function Reference

BinaryLen

Returns the number of bytes in a binary variant.

BinaryLen ( binary )

Parameters

binary The binary data to evaluate.

Return Value

Returns the length of the binary data in bytes.

Remarks

None.

Related

Binary, BinaryMid

Example

#include <Constants.au3>

Local $bBinary = Binary("0x10203040") ; Create binary data from a string.
MsgBox($MB_SYSTEMMODAL, "", "The length of the binary is: " & BinaryLen($bBinary))
