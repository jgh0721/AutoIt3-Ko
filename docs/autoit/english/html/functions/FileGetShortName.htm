Function Reference

FileGetShortName

Returns the 8.3 short path+name of the path+name passed.

FileGetShortName ( "file" [, flag = 0] )

Parameters

file full path and file name to convert
flag [optional] if 1 file can have relative dir, e.g. "..\file.txt"

Return Value

Success: Returns the 8.3 short path+name of the path+name passed.
Failure: Returns the parameter and sets @error to 1.

Remarks

The file need to exist as there is no way to known the exact ~i if several file have the same 8 first characters.

Related

FileGetLongName

Example

#include <Constants.au3>

Example()

Func Example()
    ; Create a constant variable in Local scope of the filepath to retrieve the shortname of.
    Local Const $sFilePath = @ProgramFilesDir

    ; Display the short path+name of the program files directory.
    MsgBox($MB_SYSTEMMODAL, "", FileGetShortName($sFilePath))
EndFunc   ;==>Example
