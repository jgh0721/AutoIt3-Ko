Function Reference

IsKeyword

Checks if a variable is a keyword (for example, Default).

IsKeyword ( variable )

Parameters

variable The variable to check.

Return Value

If the variable is not a keyword then the function returns zero.
If the variable is a keyword then one of the following values is returned:
Return Value Keyword
$KEYWORD_DEFAULT (1) The Default keyword.
$KEYWORD_NULL (2) The Null keyword.
Constants are defined in Constants.au3

Remarks


Related

Default, Null, VarGetType

Example

#include <Constants.au3>

Local $sDefault = Default
If IsKeyword($sDefault) Then
    MsgBox($MB_SYSTEMMODAL, "", "The variable is a keyword")
Else
    MsgBox($MB_SYSTEMMODAL, "", "The variable is not a keyword")
EndIf
