Function Reference

ControlDisable

Disables or "grays-out" a control.

ControlDisable ( "title", "text", controlID )

Parameters

title The title/hWnd/class of the window to access.
text The text of the window to access.
controlID The control to interact with. See Controls.

Return Value

Success: Returns 1.
Failure: Returns 0.

Remarks

None.

Related

ControlEnable, ControlHide, ControlCommand

Example

Example()

Func Example()
    ; Run Notepad
    Run("notepad.exe")

    ; Wait 10 seconds for the Notepad window to appear.
    Local $hWnd = WinWait("[CLASS:Notepad]", "", 10)

    ; Disable the edit control of Notepad using the handle returned by WinWait.
    ControlDisable($hWnd, "", "Edit1")

    ; Wait for 2 seconds.
    Sleep(2000)

    ; Enable the edit control of Notepad using the handle returned by WinWait.
    ControlEnable($hWnd, "", "Edit1")

    ; Wait for 2 seconds.
    Sleep(2000)

    ; Close the Notepad window using the handle returned by WinWait.
    WinClose($hWnd)
EndFunc   ;==>Example
