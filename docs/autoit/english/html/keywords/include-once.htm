Keyword Reference

#include-once

Specifies that the current file may only be included once.

#include-once

Parameters

None.

Remarks

If you include the same file containing a user-function more than once, you will get a "Duplicate function" error. When writing an include file that may be used in this way, make sure that the top line contains #include-once to prevent that file from being included more than once.

Related

#include

Example

; Use #include-once when creating UDFs so the file isn't included multiple times
; if you call the same UDF in multiple files.

#include-once
#include <Constants.au3>

; Rename this file as IncludeFunc.au3 and place next to the main script.
; Then add #include "IncludeFunc.au3" at the top of the script.

Func IncludeFunc()
    MsgBox($MB_SYSTEMMODAL, "", "This is an example of including a file.")
EndFunc   ;==>IncludeFunc
